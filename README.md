# Archivim

A gorgeous desktop application (PWA) wrapper for YouTube Music and SoundCloud built with Deno and Edge App Mode.

- **Fast & Minimal:** Powered by `yt-dlp` and `Deno` backend.
- **Customizable:** Fully customizable format, quality, metadata embedding, cover arts, cookies, proxy and proxy settings.
- **Native Experience:** Operates as a native edge window application seamlessly. 
- **Persisted State:** Uses local storage and custom Edge profile to ensure application state perfectly persists across reboots.

## Requirements
- Deno
- yt-dlp
- Microsoft Edge

## Run
`wscript.exe archivim.vbs`
